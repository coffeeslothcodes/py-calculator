# Simple CLI Calculator

A basic CLI based calculator written in Python including advanced and scientific features like finding square root, power, sin, cos, tan, natural log, etc.

***

## Features

This calculator supports the following 10 mathematical operations:

1.  **Addition** (`+`)
2.  **Subtraction** (`-`)
3.  **Multiplication** (`*`)
4.  **Division** (`/`) (Includes **division by zero** error handling)
5.  **Power** (`x^y`)
6.  **Square Root** (`âˆšx`) (Includes error handling for **negative input**)
7.  **Sine** (`sin(x)`)
8.  **Cosine** (`cos(x)`)
9.  **Tangent** (`tan(x)`)
10. **Natural Logarithm** (`ln(x)`) (Includes error handling for **non-positive input**)

***

### Requirements
Python installed on your system.
